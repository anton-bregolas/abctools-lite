ABC Tools Lite: Open Source Info

Additional third-party assets and libraries used in the project:

Mozilla's Fira Fonts are OFL-licensed fonts optimized for high legibility on screens of any size. Fira Sans and Fira Code have a wide range of glyphs, supporting 1025 languages.

Bootstrap Icons is a popular open-source SVG icon library with over 1800 glyphs designed to go well together (MIT License).

storage-available by Stijn de Witt is a short reliable script ensuring that localStorage is available and user customization data stored there can be safely applied (MIT License).

ABC Transcription Tools: Open Source Info

The converter uses open source Javascript libraries and forks of other projects:

Jens Wollschläger's ABC Transposer was the initial inspiration for the development of this tool.

abcjs by Paul Rosen and Gregory Dyke to convert abc to sheet music (MIT License).

html-to-image is used to prepare the html sheet music for PDF conversion (MIT License).

jsPDF is used to create the PDF (MIT License).

pdf-lib is used to split PDF tunebooks into individual PDF files (MIT License).

Tin Whistle Fingering Font by Paul Merke is used to generate the whistle tabs (CC BY-NC-SA 4.0 License).

qrcode.js is used to generate the QR codes (MIT License).

lz-string.js is used to for LZW compression of the tunes for share links (MIT License).

pako.js is used to for Deflate compression of the tunes for share links (MIT License).

xml2abc.js is used to for converting MusicXML format to ABC.

abc2xml is used for the ABC format to MusicXML transcoding Python web service.

Music21 (BSD 3-Clause License) is used for the MIDI to MusicXML transcoding Python web service.

jszip.js is used to unzip compressed .mxl files (MIT License).

lame.js is used to for converting .wav format to .mp3 (LGPLv2.1 License).

unmute.js is used to for fixing the iOS tab switch audio mute issue (MIT License).

MuseScore General is the basis for the rendered MuseScore soundfont (MIT License).

midi2abc is used for the standalone MIDI to ABC transcoding (MIT License).

bww2abc by Jean-Francois Moine is used for BWW to ABC conversion (GPLv3+ License).

Title Caps by John Resig is used for MusicXML title generation (MIT License).

DayPilot Modal is used for modal dialog replacements (Apache License 2.0).

context-menu is used for dropdown menus (MIT License).

ocarina_tabs by panzi (OFL License) was the starting point for the 12-Hole Ocarina Tab Creator tool.

Hiawatha's Recorder Fingering Font is used for Baroque Recorder fingering tablature.

Versilian Community Sample Library Baroque Recorder (CC0) samples were the starting point for the SATB Recorder instruments.

Ui icons created by PDF and Karacis - Flaticon.

Options icon by Marie Van den Broeck from Noun Project (CC BY 3.0)

Anglo Concertina fingering solution and ABC parsing methods used by the tab injectors originally developed by Jim Van Donsel.

Diatonic harmonica mapping algorithm derived from: abc2harp by Gek Siong Low.

FolkFriend.app Tune Collection by Tom Wyllie, used for the search engine, used under GPLv3 licensing.

Bodhran Samples by Bosone used under CC BY-NC 4.0 licensing.

Irish Bouzouki (Kontakt Sample Instrument) by Eamon Coughlan (sampled from an instrument built by Daniel Hoban) used as the basis for the Irish Bouzouki 2 instrument.

Mountain Dulcimer by Jeff Rocchio is used as the basis for the Mountain Dulcimer (Solo) and Mountain Dulcimer (DAD) instruments.

The first 30 seconds of each of the bodhran backing track YouTube videos from Santiago Molino are used as the basis for bodhran backing track custom instrument.

UltraBox Sample Extractor is the .sf2 soundfont .wav sample extractor referenced in the User Guide and cloned to Michael Eskin’s web hosting (MIT License).

Arachno Soundfont used under a variety of non-commercial use licenses.

Google Noto Sans Traditional Chinese and Noto Serif Traditional Chinese Fonts for PDF export used under the SIL OPEN FONT LICENSE Version 1.1.

CodeMirror 5 is used for the textarea replacement with line numbers and ABC syntax highlighting (MIT License).

Without these free open-source resources this project would not have been possible.
